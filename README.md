| 週 | 期間 | 学習内容 |
|----|------|----------|
| Week 1 | 〜6/22 | Git・GitHub入門、リポジトリ管理、ブランチ・マージ |
| Week 2 | 6/23〜6/29 | GitHub Codespaces、OSS貢献、プロジェクト管理(Issues/Projects) |
| Week 3 | 6/30〜7/2 | セキュリティ基礎、復習・模擬問題 |

### GH-200: GitHub Actions

Microsoft Learn パス: [GitHub Actions](https://learn.microsoft.com/ja-jp/collections/n5p4a5z7keznp5)

| 週 | 期間 | 学習内容 |
|----|------|----------|
| Week 1 | 〜6/22 | ワークフロー基礎、YAML構文、トリガー・イベント |
| Week 2 | 6/23〜6/29 | カスタムアクション作成、CI/CDパイプライン、シークレット管理 |
| Week 3 | 6/30〜7/2 | セルフホステッドランナー、復習・模擬問題 |

### GH-100: GitHub Admin（後日計画）
